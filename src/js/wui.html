WUI

Widget User Interface
based on Prototypejs.org

WUI.AutoComplete

Very simple example, an auto complete field
Code
<input type="text" id="token" name="token" />
<script type="text/javascript" charset="utf-8">
    new WUI.AutoComplete('autoComplete', {data: countries_items});
</script>

WUI.Token

Complexe example, with customized results listitem. Use the backspace or double click to remove tokens
Code
<input type="text" id="token" name="token" />
<script type="text/javascript" charset="utf-8">
    var selected = new Array();
    selected.push(168)
    var template = new Template('<span style="float:right; color:grey">#{tool}</span><div>#{label}</div>');
    var token = new WUI.Token('token', {
        data: countries_items,
        selected: selected,
        rowTemplate: template,
        onAdd: function(id) {
            $('tokenEventResults').update(e);
        },
        onRemove: function(id) {
            $('tokenEventResults').update(e);
        }
        });
</script>
